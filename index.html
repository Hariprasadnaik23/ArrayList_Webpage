ArrayList Summary

Required Packages

import java.util.ArrayList;

Syntax of Creating an ArrayList

ArrayList<Type> name = new ArrayList<>();

Example of Creating an ArrayList

ArrayList<Integer> list = new ArrayList<>();

Common Functions Used in ArrayList

1. size() - Returns the number of elements in the list

This method is used to get the size of the ArrayList.

// Example
import java.util.ArrayList;

public class Example {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        list.add(10);
        list.add(20);
        System.out.println("Size of the list: " + list.size());
    }
}
// Output: Size of the list: 2

2. add(element) - Adds an element to the list

This method adds an element to the end of the ArrayList.

// Example
import java.util.ArrayList;

public class Example {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        list.add(10);
        list.add(20);
        System.out.println("List: " + list);
    }
}
// Output: List: [10, 20]

3. add(index, element) - Inserts an element at the specified index

This method adds an element at a specific position in the ArrayList.

// Example
import java.util.ArrayList;

public class Example {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        list.add(10);
        list.add(20);
        list.add(1, 15);
        System.out.println("List: " + list);
    }
}
// Output: List: [10, 15, 20]

4. get(index) - Returns the element at the specified index

This method fetches the element located at the given index.

// Example
import java.util.ArrayList;

public class Example {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        list.add(10);
        list.add(20);
        System.out.println("Element at index 1: " + list.get(1));
    }
}
// Output: Element at index 1: 20

5. set(index, element) - Replaces the element at the specified index

This method updates the element at a particular index with a new value.

// Example
import java.util.ArrayList;

public class Example {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        list.add(10);
        list.add(20);
        list.set(1, 30);
        System.out.println("List after update: " + list);
    }
}
// Output: List after update: [10, 30]

6. remove(element) - Removes the specified element from the list

This method removes the first occurrence of the element from the ArrayList.

// Example
import java.util.ArrayList;

public class Example {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        list.add(10);
        list.add(20);
        list.remove(Integer.valueOf(10));
        System.out.println("List after removal: " + list);
    }
}
// Output: List after removal: [20]

Example Problem: Move All Zeros to the End

This problem involves moving all zeros in an ArrayList to the end of the list while maintaining the order of other elements.

// LeetCode Example

                package ArrayList_Examples;

                import java.util.ArrayList;
                import java.util.Collections;
                
                public class Medium_Question3 {
                    public static void main(String args[]) {
                        ArrayList<Integer> list = new ArrayList<>();
                        Collections.addAll(list, 0, 23, 22, 0, 5, 0, 9, 22, 11, 23, 0);
                        int index = 0;
                        for (int i = 0; i < list.size(); i++) {
                            if (list.get(i) != 0) {
                                list.set(index, list.get(i));
                                index++;
                            }
                        }
                        for (int i = index - 1; i < list.size(); i++) {
                            list.set(i, 0);
                        }
                        System.out.println("list after moving zeros: " + list);
                    }
                }
                
// Output: List after moving zeros: [23, 22, 5, 9, 22, 11, 0, 0, 0, 0, 0]
